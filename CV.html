Projet de synthèse de données - Système de distribution d'eau

Date : Décembre 2024

Contexte : Projet d'analyse et visualisation de données issues d'un réseau de distribution d'eau

Architecture du projet :

  • Pipeline de traitement : Script Shell orchestrant l'ensemble du processus d'analyse (extraction, transformation, chargement)
  • Programme C : Moteur de calcul pour le traitement intensif de grandes quantités de données
  • Visualisation : Génération automatique d'histogrammes et de graphiques statistiques

Fonctionnalités clés :

  • Calcul précis des pertes d'eau dans le réseau (fuites, évaporation, erreurs de mesure)
  • Analyse statistique des flux par segment de réseau
  • Détection d'anomalies et points critiques
  • Export des résultats en formats CSV et graphiques

Optimisations techniques :

  • Structures AVL : Implémentation d'arbres équilibrés pour recherches et insertions en O(log n)
  • Gestion mémoire efficace : Allocation dynamique optimisée, libération systématique, profiling avec Valgrind
  • Traitement par batch : Gestion de fichiers volumineux sans saturation mémoire

Technologies : C (calcul), Bash (automatisation), Gnuplot (visualisation)

Compétences mobilisées : Structures de données avancées, scripting système, analyse de données, optimisation algorithmique
